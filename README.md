# Student Admission Information Portal (HTML Only)

A simple multi-page static website built using pure HTML only (no CSS, no JavaScript), designed to provide information about the admission process of a college.

This website demonstrates:
- HTML structure and formatting
- Hyperlink navigation across multiple pages
- Forms, tables, lists, images & semantic tags
- Clear and organized content relevant to student admissions

## Project Features

- Well-structured HTML Pages  
- Fully interconnected navigation  
- Admission Process details  
- Courses Offered (Table Format)  
- Contact Information  
- Student Enquiry Form  
- Demonstrates usage of multiple HTML tags like:
-`<table>`, `<form>`, `<input>`, `<textarea>`, `<img>`, `<address>`, `<ul>`, `<ol>`, `<footer>`, etc.
